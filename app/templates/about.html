Acerca del proyecto

Este proyecto utiliza Flask, un microframework web en Python para crear aplicaciones web dinámicas.

Dependencias y modulos

La finalidad de este proyecto es hacer uso de lo aprendido en clases adicionalmente se han estudiado otros modulos con el fin de aprender y practicar las bases para contruir aplicaciones web dinámicas.

Se pretende utilizar, SQLAlchemy como ORM para conectarse a una base de datos MySQL. Para hacer uso de Flask-SQLAlchemy se ha utilizado la extensión flask_sqlalchemy.

Para el manejo del enrutamiento se utiliza blueprints

Para el manejo de formularios será utilizado Flask-WTF.

En cuanto al frontend de momento se utilizan plantillas de jinja2 para hacer uso de los templates. Se desea hacer uso de tailwind para mejorar la presentación del proyecto.
